# minitest-activemodel

MiniTest matchers for ActiveModel::Validations.

## Installation

Add this line to your application's Gemfile:

    gem 'minitest-activemodel'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install minitest-activemodel
